Kéo thả file Excel hoặc Nhấp để chọn

Chỉ hỗ trợ file định dạng .xlsx, .xls

Cấu trúc cột Excel Tối thiểu:

  • Cột A (Index 0): Mã học sinh (Ví dụ: HS2025001)
  • Cột B (Index 1): Tên môn học (Ví dụ: Toán)
  • Cột C (Index 2): Điểm số (Ví dụ: 8.5)
  • Cột D (Index 3): Loại điểm (Tùy chọn, mặc định: Miệng)
  • Cột E (Index 4): Học kỳ (Tùy chọn, mặc định: HK1)

Lưu ý: Bỏ qua dòng số 1 (dòng Header).
